# `incli user`

### Add a user

To add a new user, use the `add` subcommand:

```bash
incli user add pete
```

### Delete a user

To delete a user, use the `delete` subcommand:

```bash
incli user delete pete
```

### List users

To list the _Auth_ server's known users, use the `list` subcommand:

```bash
incli user list
```
